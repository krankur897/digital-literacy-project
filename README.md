## 📝 Task Summary

### 🔹 Task 1: Digital Literacy Infographic
I created an infographic using **Canva**, covering topics like digital safety, online tools, and responsible behavior on the internet.

### 🔹 Task 2: Digital Portfolio
I built and improved my profiles on **GitHub, LinkedIn, and HackerRank**, focusing on creating a professional online presence.

### 🔹 Task 3: Coding & Collaboration
This task involved solving a coding problem and creating a **Google Form quiz** to spread awareness about digital literacy.

### 🔹 Task 4: Email Etiquette
I practiced writing formal and professional emails and also created a checklist for responsible social media usage.

### 🔹 Task 5: Cybercrime Awareness
I studied common cyber threats like phishing and online fraud and learned how to prevent them.
